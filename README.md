# E-commerce Sales Analysis

This project analyses an e-commerce sales dataset from Kaggle using Python. The goal of the project is to clean the dataset, explore sales patterns, and answer business-related questions using data analysis and visualization.

## Questions Answered
- How does revenue change across the months?
- Which products generate the most revenue, and which products are sold the most?
- At what time of day are most orders placed?
- Which cities generate the highest revenue?
- Which products are most frequently ordered together?

## Methods Used
- Data cleaning
- Datatype conversion
- Feature creation (`Sales` column)
- GroupBy analysis
- Data visualisation with Matplotlib

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
This dataset was sourced from [Kaggle](https://www.kaggle.com/code/zainabsiddiqui2322/ecommerce-sales-analysis).
